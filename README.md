# How to run it : 

## 1- Create a virutal environment : 

### For Linux users : 

python3 -m venv env

source env/bin/activate

### For Window users (in cmd) : 

python3 -m venv env

env\Scripts\activate

## 2 - Install requirements :

pip install -r requirements.txt

## 3 - Run it ! :

python3 test.py
